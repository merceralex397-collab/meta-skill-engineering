# Archive

These skill packages were archived because they exist primarily for external distribution workflows (packaging, installing from registries, provenance auditing, community harvesting) rather than for internal skill creation, improvement, and evaluation.

They remain here as reference material. If concepts from these skills are needed, fold them into the active skill inventory rather than reactivating the archived package.

## Archived Skills

| Skill | Reason |
|-------|--------|
| `community-skill-harvester` | Finds and imports skills from external registries and public repos. Internal skills are authored in this repo. |
| `skill-installer` | Installs skill packages from GitHub/archives into local agent client directories. Internal skills are already in the repo. |
| `skill-packaging` | Bundles skills into versioned archives with manifests, checksums, and client-specific overlays for distribution. |
| `skill-provenance` | Audits origin, authorship, license, and trust level for externally-sourced skills. Internal skills have known provenance. |
