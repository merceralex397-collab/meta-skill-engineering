# Library — Verified Skills

Skills in this directory have been tested, evaluated, and approved for production use.

## How skills arrive here

1. **Unverified** (LibraryUnverified/) — Raw or imported skills, not yet tested.
2. **In Testing** (LibraryWorkbench/) — Skills undergoing evaluation and benchmarking.
3. **Verified** (Library/) — Skills that passed quality thresholds and are ready for use.

Use Meta Skill Studio to promote skills through these tiers.
